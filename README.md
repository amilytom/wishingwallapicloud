# wishingwallapicloud
许愿墙REST API接口（node+express+mysql）

### 项目使用以下技术栈

-   async                   异步处理方法库
-   mysql2                MySQL数据库支持
-   sequelize            MySQL映射
-   dateformat        时间处理方法库
-   jsonwebtoken   Token生成及验证
-   art-template  express-art-template      art-template 模板引擎
-   nodemon           node运行自动化监控工具

### 项目运行

下载项目文件夹后，先安装依赖

```
npm install
```

然后

```
npm start
```

即可。

因为是API接口，所以浏览器是无法访问的。

使用Postman或者ApiPost工具来测试api

API网址是 http://localhost:1688
